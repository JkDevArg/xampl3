QR Scan - Aplicación CiberSeguridad | OpenSource

Aplicación creada con NestJS (Backend) y ExpoJS (Frontend). Escanea de forma segura los QR identificando si es fraude o un enlace sospechoso.

Propósito y alcance

QRSec permite a los usuarios:

Escanear códigos QR con la cámara de su dispositivo Validar las URL contenidas en los códigos QR con un backend(NestJS) de seguridad Recibir advertencias sobre URL potencialmente maliciosas Abrir URL validadas de forma segura La aplicación está desarrollada con React Native y el framework Expo, lo que proporciona una solución multiplataforma para dispositivos iOS y Android.

Flujo

La aplicación QRSec sigue una arquitectura cliente-servidor donde la aplicación móvil escanea códigos QR, extrae URL y las envía a un servicio de validación backend. Según la respuesta, la aplicación advierte al usuario sobre una URL potencialmente maliciosa o ofrece la opción de abrir una URL segura.

Permisos y configuración

QRSec requiere permisos de cámara para funcionar correctamente. Estos permisos se configuran en el archivo app.json:

Descripciones de uso de la cámara y el micrófono para iOS Permisos de cámara y grabación de audio para Android Plugins esenciales de Expo para enrutamiento, pantalla de inicio y escaneo de códigos de barras

Tecnología

QRSec se desarrolla utilizando las siguientes tecnologías:

Framework Expo (React Native) Cross-platform mobile development framework NestJS Backend framework for Node.js MySQL Database for data storage
